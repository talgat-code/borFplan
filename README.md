# PlanBot

Телеграм-бот для планирования дел: задачи на день/неделю, приоритеты, матрица
Эйзенхауэра, напоминания и утренняя сводка. Данные хранятся локально в SQLite.

## Возможности

- 📅 Дела на сегодня / завтра / ближайшую неделю
- 📊 Матрица Эйзенхауэра (важно/срочно)
- ⚠️ Просроченные дела и ⏰ напоминания
- 📈 Статистика и поиск по делам
- 🌅 Утренняя сводка дел

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

Создайте файл `.env` с токеном бота (от [@BotFather](https://t.me/BotFather)):

```
BOT_TOKEN=ваш_токен
PLANBOT_TZ=Asia/Almaty   # необязательно, по умолчанию Asia/Almaty
```

Затем:

```bash
python bot.py
```

В VS Code бот также запускается одной кнопкой **Run** (см. `.vscode/launch.json`).

## Команды

| Команда | Назначение |
|------------|-----------------------------------|
| `/today`   | Дела на сегодня |
| `/tomorrow`| Дела на завтра |
| `/week`    | На ближайшую неделю |
| `/all`     | Все будущие дела |
| `/day`     | Дела на конкретный день |
| `/overdue` | Просроченные дела |
| `/matrix`  | Матрица Эйзенхауэра |
| `/find`    | Поиск по делам |
| `/stats`   | Статистика |
| `/done`    | Отметить дело выполненным |
| `/del`     | Удалить дело |
| `/prio`    | Изменить приоритет |
| `/edit`    | Редактировать дело |
| `/snooze`  | Отложить дело |
| `/remind`  | Поставить напоминание |
| `/morning` | Настроить утреннюю сводку |
| `/clear`   | Очистить выполненные |
| `/reset`   | Сбросить данные |

## Стек

- Python 3.11+
- [python-telegram-bot](https://python-telegram-bot.org/) 21.9
